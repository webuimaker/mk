APROVECHALOS - CARIBE MEDIA INC - ONLINE DEALS

Concept, Microsite, Mobile Application

  • The Brief
    The website should have the following deal types: Daily Deals (Normal Deals), Products and Getaways. Additionally, the website should have a Gift Cards and Voucher Section in which the user can select the purchase value of the gift card or voucher for the participating businesses. Platform should include an Administrator Module and a Merchant Module. Mobile website and apps for iPhone and Android are desirable.
  • My Role
    With tight schedule of project, I got the chance to design this e-commerce (40+ pages) online deals portal including admin panel which combines the latest e-commerce technology. In the saturated world of e-commerce, it is important to have a solid brand message and visual identity that span across all online touch points.
